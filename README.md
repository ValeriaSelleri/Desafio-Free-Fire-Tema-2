# Desafio Free Fire – Tema 2
Trabalho de Estrutura de Dados — Estácio

Este repositório contém uma implementação base (C + versão em Python) para o Tema 2:
- Modelagem de jogadores e itens.
- Distribuição inicial de itens (round-robin).
- Listagem do estado inicial.

## Como compilar (C)
Entre na pasta `src` e execute:
```bash
make
./freefire
```

## Como executar (Python)
```bash
python3 freefire_tema2.py
```

## Arquivos
- `src/` : código em C (main.c, jogadores.c/h, itens.c/h, Makefile)
- `mapa.txt` : exemplo de mapa (na raiz)
- `freefire_tema2.py` : versão em Python
